# 🗳️ WANTAM — One Term. You Promised.

> *A reminder. A movement. A very mild inconvenience.*

---

## What is this?

**Wantam** is a satirical, humorous static website inspired by the Kenyan civic concept of "one term" — the idea that leaders should honour their word and step down after a single term. The name is a phonetic blend of "one term."

This project is built in the spirit of sites like [The Useless Website](https://theuselessweb.com/) — a series of absurd, looping interactive scenes that poke fun at the universal human tendency to overstay a welcome. No databases. No backend. Just vibes and JavaScript.

## 🚀 Running Locally

No setup needed. Just open the file:

```bash
# Clone the repo
git clone https://github.com/yourusername/wantam.git

# Open in browser
open index.html
# or just double-click index.html in your file explorer
```

Requires an internet connection only for Google Fonts to load. Everything else works offline.
